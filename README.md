# 第一章 无线网络概述

## 基本概念

- [x] 无线通信(Wireless Communication)
  - `广义` ：没有“线”的通讯，甚至不需要物理传输媒介；
    > - 心灵感应、量子之间的纠缠态等；
  - `狭义` ：利用 `电磁波` 信号可以在自由空间中传播的特性进行信息交换的一种通信方式。

- [x] 无线网络(Wireless Networks)
  - 采用无线通讯技术组建的网络，包括：
    > - 允许用户建立远距离无线连接的 `全球语音和数据网络` ；
    > - 为近距离无线连接进行优化的 `红外线技术及射频技术` 。
